# ME:AM

**Record your voice at night. Wake up to your own motivation in the morning.**

ME:AM is a self-motivation alarm app. The idea is simple: every night, you record a motivational message to yourself. The next morning, your own voice wakes you up — not a generic ringtone, but *you* telling *you* why today matters.

## Features

- **Voice Recording** — Record, rename, play, favorite your motivational messages
- **Custom Alarms** — Set alarms with your own recordings as the sound
- **Shuffle Mode** — Wake up to a random favorite recording each morning
- **Time Capsules** — Send a voice message to your future self, delivered as an alarm
- **Repeating Alarms** — Daily, weekdays, or custom day selection
- **Snooze & Vibration** — Configurable snooze duration and vibration patterns
- **Pixel Art UI** — Retro game aesthetic with animated mascot, waveforms, and starfield background

## Tech Stack

- React Native + Expo (SDK 54)
- TypeScript
- Pixel art assets generated with [Pixellab](https://pixellab.ai)
- Press Start 2P font

## Run Locally

```bash
cd ME.AM
npm install
npx expo start
```

Scan the QR code with Expo Go (iOS/Android) or press `w` for web.
